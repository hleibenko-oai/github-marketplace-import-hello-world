# Trail Docs

Use AllTrails to research trail options and Notion to create a practical guide for the selected hike.

## Workflow

1. Ask for any missing constraints that materially affect the search, such as location, date, difficulty, maximum distance, or group needs.
2. Use the AllTrails app to find suitable hikes. Compare a small set of strong options using available details such as distance, elevation gain, expected duration, rating, and highlights.
3. Recommend one trail and explain why it best matches the user's constraints.
4. When the user asks to publish the plan, use the Notion app to create a shareable page with:
   - a concise trail summary
   - route details and the AllTrails link
   - a packing checklist
   - a meetup plan
   - any caveats or open questions
5. Return the Notion page link and briefly state what was included.

If either app is unavailable, tell the user which app needs to be connected and continue with any useful partial result the available app supports.
